## Installation with docker-compose on Ubuntu 20.4 

Web requires version[docker]>= 20.10.17 and version[docker-compose] >= 1.27.0
```sh
sudo su
git clone https://github.com/LNOI/nOke_nodejs
cd nOke_nodejs
docker-compose up --build
http://127.0.0.1
```
